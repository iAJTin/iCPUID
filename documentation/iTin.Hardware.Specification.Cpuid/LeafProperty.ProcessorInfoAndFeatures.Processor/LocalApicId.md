# LeafProperty.ProcessorInfoAndFeatures.Processor.LocalApicId property

Gets a value representing the key to retrieve the property value.

Local APIC ID. The initial APIC-ID is used to identify the executing logical processor. This number is the 8-bit ID that is assigned to the local APIC on the processor during power up. This field was introduced in the Pentium 4 processor. It can also be identified via the [`x2APICID`](../LeafProperty.ExtendedTopologyInformation/x2APICID.md).
